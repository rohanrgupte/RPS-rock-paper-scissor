# Rock Paper Scissor - AI Opponent

AI-powered Rock Paper Scissor game using Multi-Armed Bandit with Random Forest.

## Demo

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-name.streamlit.app)

## Quick Start

```bash
pip install -r requirements.txt
streamlit run app.py
```

## How It Works

- Uses Random Forest to learn player patterns
- Adapts strategy using Multi-Armed Bandit approach
- Predicts and counters user moves

## Tech Stack

- Python
- Streamlit
- scikit-learn
- NumPy
